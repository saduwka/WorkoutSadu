# SADU

iOS‑приложение **SADU**: тренировки, здоровье, задачи, финансы, привычки. SwiftUI, SwiftData. Виджеты, Share Extension для чеков, ИИ‑чат (Life Bro).

- **Табы:** Сегодня, Здоровье (тренировки + еда/калории), Задачи (привычки, todo, цели), Деньги, Я (профиль, квесты, рекорды, отчёты).
- **Документация:** [CHANGELOG.md](../CHANGELOG.md) — история изменений по версиям. [LifeOS_Roadmap.md](../LifeOS_Roadmap.md) — roadmap и архитектура.
